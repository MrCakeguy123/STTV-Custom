STTV Custom

A custom build of SmartTwitchTV
v2026.03.20

This is a modded version of SmartTwitchTV with re-enabled ad-blocking, automatic proxy failover, a live debug overlay, and Firebase/Crashlytics removed for privacy.

Download Original Project

What's Changed

Everything in this build that's different from the original SmartTwitchTV by fgl27.

Ad-Block

Re-enabled proxy-based ad-blocking

The original project had proxy/ad-blocking code commented out. This build re-enables it with TTV LOL Pro as the default proxy, using the updated endpoint (lb-eu.cdn-perfprod.com).

Ad-Block

Proxy auto-fallback chain

When a proxy fails 3 times in a row, the app automatically rotates to the next one. The chain cycles through TTV LOL Pro, K-Twitch, and T1080 so you don't have to manually switch when one goes down.

Feature

Ad-block debug overlay

A toggleable on-screen overlay that shows real-time stats: proxy status, proxy ping, requests proxied vs. direct, ad-block success rate, player bitrate/buffer/dropped frames/latency, and a fallback event log. Enabled by default in Settings > Custom.

Feature

Quick proxy switcher in player controls

The proxy control that was commented out in the player's bottom bar has been restored. Switch between proxies or disable them without leaving the player.

Feature

Custom settings section

A new "Custom" section in Settings for mod-specific options: ad-block debug overlay toggle and proxy auto-fallback toggle.

Privacy

Firebase & Crashlytics removed

All Firebase, Google Analytics, and Crashlytics code has been disabled. No crash reports, no analytics, no data sent to Google. Error logging stays local.

Feature

Custom branding & version system

The About screen and version display show "STTV Custom" branding with a separate version tag, making it easy to tell this build apart from the original. Changelog includes a dedicated entry for this fork's changes.

Feature

Loads from local assets

The APK loads the web app from bundled local assets instead of downloading from GitHub Pages. This means faster startup and no dependence on an external server for the UI.

Original Project

SmartTwitchTV is created and actively maintained by fgl27. The core application — the web UI, the Android APK, the Twitch API integration, the player, chat, settings, and everything else — is their work. This fork only modifies a small portion of the codebase to re-enable ad-blocking features and add some extra tools.

The project was originally started by nikitakatchik and later forked by CazuzaCzz before fgl27 rebuilt it from scratch.
